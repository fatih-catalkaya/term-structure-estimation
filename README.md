Zum Aufsetzen des Projektes die folgenden Befehle der Reihe nach ausführen
```shell
# Unter Windows
python -m venv venv
.\venv\Scripts\activate.ps1
pip install -r requirements.txt

# Unter Linux
python -m venv venv
pip install -r requirements.txt
```
Danach kann das Projekt mit `python main.py` ausgeführt werden
